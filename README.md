仿简书单页应用，主要技术栈包含<br>
（React、react-router、react-redux、immutable、redux-thunk、styled-components、axios、react-loadable）等。

## 运行此项目

### 第一步：下载

### `git clone https://github.com/Laule/jianshu`

### 第二步：安装

### `npm install`

### 第三步：启动

### `npm run start`

在开发模式下运行应用程序.<br>
打开浏览器输入 [http://localhost:3000](http://localhost:3000)
